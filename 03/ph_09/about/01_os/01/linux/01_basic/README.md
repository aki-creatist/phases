# システムの停止

* `shutdown` コマンド
    * システムを停止したり再起動したりする

```bash
shutdown [オプション] [実行時間]
```

* オプションにより、動作と実行時間を指定できる
    * オプションを指定しないと、シングルユーザーモードというメンテナンスモードに入る
    * shutdownコマンドを実行できるのは管理者ユーザーだけ

## 動作に関するオプション

| オプション | 説明 |
|:----|:----|
| h | システムを停止する |
| r | システムを再起動する |

## 時間に関するオプション

| 実行時間 | 説明 |
|:----|:----|
| now | 今すぐ |
| +5 | 5分後 |
| 21:00 | 21時 |

例: 今すぐシャットダウンする

```bash
shutdown -h now
```
