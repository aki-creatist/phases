# Bootstrapのレイアウト

* レイアウトを12分割してグリッドレイアウトを構成する
* 12個のブロックを自由に組み合わせることで、複雑な構成も簡単に実現できる

## 手順

* 各行を`<div class="row">タグ`で囲む
* その中に列ブロックとして`<div class="col col-md-数字">タグ`を記述
    * 数字の合計が`12`になるように構成する
