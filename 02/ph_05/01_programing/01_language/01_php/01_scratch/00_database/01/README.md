## ヒアドキュメント

## 背景

* 処理によってはSQL文がかなり長いものになることがある
* このようなときは、ヒアドキュメント構文を使うと見やすく間違いが少ない記述が可能

## 概要

* `<<<EOS`を記述して、`EOS;`で終わる
* 変数の格納方法この`EOS`には意味はなく、任意の文字列をつけることが可能
    * ただし、英数字と` _ `アンダーバーのみ
    * 数字から始まる文字はエラーになる
    * はじめの`<<<`と`EOS`の間は空けない
    * 最後のEOSの末尾は`;`のみ
    * `EOS;`以降に半角スペースなど文字列があるとエラーになる
    * `EOS;`の前にもタブやスペースがあってはならない
    * なお、PHP上で発行するSQL文の最後は`;`で終わる必要はない

```php
$sql = <<<EOS
INSERT INTO member
    (
        last_name,
        first_name,
        age
    )
VALUES
    (
        '田中',
        '一郎',
        21
    )
EOS;
```
