# 画面機能パターン

* 画面機能パターンでは、機能に応じて、画面レイアウトにおけるボディの構成と画面遷移をパターンとして検討
    * 検索画面、一覧画面、登録・更新画面etc

## 画面機能パターンの例

* [検索画面パターン](01_search)
    * 情報の検索条件を入力する画面のパターン
* [一覧画面パターン](02_index)
    * 情報を一覧表示する画面のパターン
* [登録・更新画面パターン](03_create)
    * 情報を登録・更新する画面のパターン
    * 確認画面と完了画面を含める
* [削除画面パターン](04_confirm)
    * 情報を削除する画面のパターン
    * 確認画面と完了画面を含める
