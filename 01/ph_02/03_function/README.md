# 機能について考えよう

## 機能とは

* 機能 = ファンクション = 関数 = 仕事

一般的には以下のように表される

```text
y = f(x)
```

* xが引数でyが戻り値
* xを入力、yを出力ととらえることもできる
* xを材料、yを成果と考えることもできる
    * 機能とは仕事

```text
成果　＝　活動(材料)
```
