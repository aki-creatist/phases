# 情報システム化の流れ

* 経営層が`経営戦略の策定`を行う
* 経営戦略の策定の結果を受け、業務改革の領域で`BPR`を行う
    * Business Process Reengineering / BPM (Business Process Management)
* IT活用領域でコンピュターシステムを構築
* 結果評価
    * 評価や効果で経営層は情報システム化に対する次の投資の意思決定
        * 評価: 企業に対する評価
        * 効果: 企業の業績向上など
